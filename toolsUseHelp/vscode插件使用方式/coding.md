1. 代码调试
   1. Turbo Console Log 
      1. 快速添加 console.log 信息，js debug 必备
      2. 使用方式
         1.  ctrl + alt + l 选中变量之后，使用这个快捷键生成 console.log
         2. alt + shift + c 注释所有 console.log
         3. alt + shift + u 启用所有 console.log
         4. alt + shift + d 删除所有 console.log
